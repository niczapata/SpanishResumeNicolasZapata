## Contents

- `CVNicolasZapataAlzate.tex` - Main LaTeX source file

## Compilation

To compile the resume, ensure you have a LaTeX distribution installed (TeX Live, MiKTeX, or MacTeX) with the following packages:
- `babel` (with english and spanish options)
- `geometry`
- `enumitem`
- `hyperref`
- `xcolor`
- `fontawesome5`
- `multicol`
- `titlesec`
- `parskip`
- `graphicx`
- `setspace`

Run the following command:

```bash
pdflatex CVNicolasZapataAlzate.tex
```

- **Language**: Spanish with English support
- **Color Scheme**: Professional blue accent (RGB: 40, 100, 200)
